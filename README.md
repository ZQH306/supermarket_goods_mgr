# 超市管理系统

## 项目简介
这是一个模拟超市核心购物流程的控制台应用程序。系统实现了从商品信息管理、顾客浏览选购、购物车操作到最终结算的完整闭环，结构清晰，模块化程度高。

## 功能模块
系统严格按照购物流程设计，包含以下八个核心功能模块：

01 - 开始菜单实现：系统入口，提供所有功能的导航菜单。

02 - 建立库存商品信息：后台核心模块，用于初始化和管理所有商品的编号、名称、价格、库存等基础数据。

03 - 显示商品信息：向顾客展示当前可售的商品清单。

04 - 购物车菜单功能实现：提供购物车管理的操作子菜单（添加、查看、修改、删除等）。

05 - 购物车商品添加功能实现：将顾客选中的商品加入虚拟购物车。

06 - 购物车商品显示：清晰展示购物车内所有商品的详情、数量、小计及金额总计。

07 - 结算功能：完成支付流程，打印购物小票，并更新库存数据。
